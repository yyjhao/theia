Const FileShouldOverwrite

FileShouldOverwrite: unique symbol = Symbol('FileShouldOverwrite')
  • Defined in packages/filesystem/src/common/filesystem.ts:202

A callback type, called when we try to save a file but realize it has been modified by somebody else since we have opened it. originalStat is the stat at the moment we opened the file, currentStat is the stat at the moment we try to save it (after the external modification). The callback should return true if we still want to save the file, false otherwise.
