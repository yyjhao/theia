_scope

_scope: string | undefined

Inherited from TaskConfiguration._scope

  • Defined in packages/task/src/common/task-protocol.ts:108

For a provided task, it is the string representation of the URI where the task is supposed to run from. It is undefined for global tasks. For a configured task, it is workspace URI that task belongs to. This field is not supposed to be used in tasks.json

_source

_source: string
  • Defined in packages/task/src/common/task-protocol.ts:117

Source of the task configuration. For a configured task, it is the name of the root folder, while for a provided task, it is the name of the provider. This field is not supposed to be used in tasks.json
