Optional alignment

alignment: Alignment
  • Defined in node_modules/@phosphor/widgets/lib/splitpanel.d.ts:172

The content alignment of the panel.

The default is 'start'.

Optional layout

layout: SplitLayout
  • Defined in node_modules/@phosphor/widgets/lib/splitpanel.d.ts:186

The split layout to use for the split panel.

If this is provided, the other options are ignored.

The default is a new SplitLayout.

Optional orientation

orientation: Orientation
  • Defined in node_modules/@phosphor/widgets/lib/splitpanel.d.ts:166

The layout orientation of the panel.

The default is 'horizontal'.

Optional renderer

renderer: IRenderer
  • Defined in node_modules/@phosphor/widgets/lib/splitpanel.d.ts:160

The renderer to use for the split panel.

The default is a shared renderer instance.

Optional spacing

spacing: undefined | number
  • Defined in node_modules/@phosphor/widgets/lib/splitpanel.d.ts:178

The spacing between items in the panel.

The default is 4.
